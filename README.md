# PicoSmartHub

Raspberry Pi Pico W を中核とした、多機能なスマートホームハブ（環境測定ステーション）プロジェクトです。

## 主な機能
- **環境測定**: DHT20 (温湿度)、DHT22 (屋外温湿度)、AS3935 (雷センサー)、超音波センサー連携
- **機器制御**: SwitchBot APIとの連携による家電操作
- **赤外線リモコン**: 既存の赤外線リモコン信号の送受信・学習
- **Wake on LAN**: ネットワーク経由でのPC起動
- **通知連携**: LINE Messaging APIやGoogle Apps Scriptとの連携

## 使い方
1. 本リポジトリのコード（`PicoSmartHub.ino` 等）をダウンロードします。
2. `config.h` 内の `YOUR_***_HERE` となっている箇所を、ご自身の環境に合わせて設定してください（WiFiパラメータ、各種APIキー、MACアドレスなど）。
3. Arduino IDE等を使用して Raspberry Pi Pico W に書き込みます。
